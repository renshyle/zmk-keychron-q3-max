# ZMK on Keychron Q3 Max

This is a ZMK module that introduces the `q3_max` board.

## Features

* USB
* Pressing (and releasing!) keys

That may seem like a short list of features and such it is. This module only includes the ISO layout
and lacks many features, such as the knob (except pressing it), LEDs, Bluetooth, and 2.4 GHz.
